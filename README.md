# ChatGPT-Easy
ChatGPT OpenAI API fast startup
简单的 ChatGPT 服务 API 调用

## How to install ?

```shell
git clone https://github.com/chenjinya/chatgpt-easy
cd chatgpt-easy
npm install
```

## How to use ?

![example]('./example.png')

```shell
export OPENAI_KEY='OPENAI_KEY'
npm run chat
```

## Prepare for env in CentOS 

```shell
yum install git nodejs vim
```
